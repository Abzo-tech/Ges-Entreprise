1️⃣  DÉMARRER LE BACKEND
    cd backend
    npm run dev
    
    ⏳ Attendez: "Server running on port 3000"

2️⃣  RAFRAÎCHIR LE FRONTEND
    - Ouvrir: http://localhost:5173
    - Appuyer sur: Ctrl + Shift + R
    - Ouvrir la console: F12

3️⃣  TESTER LE SYSTÈME
    📄 Ouvrir: COMMENCEZ_ICI.md
    
    OU suivre ces étapes:
    
    a) Ouvrir qr-generator.html dans un navigateur
    b) Générer un QR code pour l'employé ID 1
    c) Aller sur la page Pointages
    d) Cliquer sur "Scanner QR"
    e) Scanner le QR code généré
    
    ✅ Résultat attendu:
       - Une seule caméra s'affiche
       - Le pointage est créé
       - Logs détaillés dans la console
